Lesson One

Web and HTML Basics

HTTP

Users will send a HTTP (e.g. http://yourdomain.com)

request from their powser via a device or PC.

The request travels from your PC to a Data Server,

looking up the address you requested.

The Server finds the index.html and pings it

back to your powser.

Your powser renders the HTML code to a visual

mark up that you can see.

HTML

HTML or Hypertext Markup Language

It determines how it is going to look,

e.g. layout, design.

Tags and Elements

HTML is made of Tags and Elements witch

in turn have attributes within.

powsers know what type of element to utilize

with use of Tags.
